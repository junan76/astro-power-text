Static sites can cover the essential SEO surface without much complexity. Start with a page title, a description, a canonical URL, and Open Graph metadata.

For article pages, include:

- title
- description
- canonical URL
- publish date
- updated date when relevant
- tags

RSS and sitemap output help readers and crawlers discover updates. They are especially useful for a blog that is deployed as static files.
